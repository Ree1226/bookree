📊 ランキングの仕組み

当サイトでは、ユーザーの皆様からの投票（1〜5点）を集計し、リアルタイムでランキングを作成しています。

🏆 ポイント2倍 ブースト対象

多くの読者に支持されるべき「話題作」や「名著」を見逃さないよう、以下の条件に該当する本は点数が2倍で加算されます。

  • 👑 三大主要賞：
    本屋大賞 / 直木賞 / 芥川賞
  • 🎖 エンタメ・ジャンル賞：
    山本周五郎賞 / 吉川英治文学賞（新人賞含む） / 江戸川乱歩賞 / 星雲賞 / このミステリーがすごい！ / 三島由紀夫賞 など
  • 📺 メディア化・話題作：
    映画・ドラマ・アニメ化作品 / ベストセラー（部数突破）

※Google Booksの登録データ（説明文）に基づき自動判定しています。 そのため、受賞作であっても説明文に記載がない場合はボーナス対象外となることがあります。あらかじめご了承ください。

🔥 最新トレンドの定義

「最新トレンド」ランキングでは、新しい本との出会いを促進するため、以下の基準で作品を表示しています。

  • 📅 対象出版年：
    2025年 および 2026年に出版された本
  • 📖 データの扱いについて：
    出版年はGoogle Booksのデータに基づきます。そのため、実際の初版年と異なる場合があります。
  • 例：2022年に単行本化、2025年に文庫化された作品の場合、文庫版のデータ（2025年）としてトレンドに掲載されます。

🔍 本の検索について

「BOOK REE」では、より多くの本に出会えるよう、以下の仕組みで検索機能を提供しています。

🌐 検索対象：
世界中の書籍データが登録されている Google Books API を参照しています。

⚙️ 検索ができない場合：
外部の検索システムを利用しているため、アクセスが集中している場合やシステムメンテナンス等により、一時的に検索が行えないことがあります。その際は、恐れ入りますが少し時間を置いてから再度お試しください。

⏳ 反映タイミング：
出版されて間もない新刊は、システムへのデータ反映状況により、検索結果に表示されない場合があります。

💡 検索のコツ：
「タイトル」だけでなく、「タイトル 著者名」のようにスペースで区切って入力すると、目的の本が見つかりやすくなります。また、特定のタイトルが不明な場合でも、「ミステリー」「料理」などのキーワードや登場人物、出版社名等で検索することで、該当の本が見つかることもあります。

自動登録について：
検索で見つかった「まだランキングに載っていない本」に投票すると、その本は自動的に当サイトのデータベースに登録されます。
